# Schedule

[![License: GPL v3](https://img.shields.io/badge/License-GPLv3-blue.svg)](https://www.gnu.org/licenses/gpl-3.0)

This is small schedule app for students to keep of their daily schedule at a glance.

## Features

* Current time
* Remaining time
* Dynamic schedule changing depending on the day of the week
* Today's lunch

This project is currently very specific to my school. As I develop this project further, I will work on making this as general and customizable as possible.
